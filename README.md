Non-mewtonian cat
===

A game made for the [js13k 2025](https://js13kgames.com/) game jam.

## Build

Requirements:
- [npm](https://www.npmjs.com/)

Install dependencies:
```sh
npm install
```

Build:
```sh
npm run build
```

Dev server:
```sh
npm run dev
```

Build final zip:
```sh
npm run all
```
